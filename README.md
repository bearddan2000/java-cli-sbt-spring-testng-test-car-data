# java-cli-sbt-spring-testng-test-car-data

## Description
A POC for spring api test using TestNG.
Writes test results to console
and html.

## Tech stack
- java
- sbt
  - spring
  - TestNG

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
